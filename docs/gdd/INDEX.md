# Brambly Hedge: The Game — Game Design Document

**Aspirational title:** Brambly Hedge: The Game · **Public codename:** Project Hedgerow ([naming rules](./01-working-title.md))

This directory is the Game Design Document. Each section lives in its own file; the chain runs [00-decisions](./00-decisions.md) → 01 → … → 21. These documents are private and may use canon names freely ([D1](./00-decisions.md)); anything public-facing uses the codename only.

## Core Identity

> **"The first cozy sim where the community demonstrably runs, works, and celebrates with or without you — your reward is belonging, not credit."**
>
> *You don't build your empire. You build the community.*

## Reading Order

1. **[00-decisions.md](./00-decisions.md) first — it is binding.** Every number, name, and rule elsewhere must agree with it; where documents conflict, it wins.
2. Then [02-vision.md](./02-vision.md) and [03-design-pillars.md](./03-design-pillars.md) for the big picture.
3. [07-camera-direction.md](./07-camera-direction.md) and [18-community-philosophy.md](./18-community-philosophy.md) define the two flagship differentiators.
4. [20-mvp.md](./20-mvp.md) scopes the first playable; [production docs](#production-documents) carry the plan.
